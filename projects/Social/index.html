PROJECTS ABOUT RESUME

Social Media App

This is the biggest project I have worked in.

This app contains every feature of a social media app. The features include -

  • Making Posts
  • Connecting With Friends
  • Sharing Images and Streaming Videos
  • Messaging Service
  • Two-Player Tic-Tac-Toe (For timepassing)

Posts

Users can create and share posts with friends. You can delete your post if you want. You can share an image or a video with each post. Your friends can see you posts and like and comment on it.

Friends

You can send friend request to any user. And accept or delete a request if someone sends you a friend request. You can cancel your sent request later if you want. Once you become friend with someone, you can see their posts. This is an intentional constrain. (I will work on the UI of this page later)

Video Streaming

I integrated a video streaming service in this project that I stole from here. It lets you play a part of video in realtime without loading the whole video.

Messaging

I developed this app from the basics. I read throught the documentation of Spring Boot Websocket. It was fun to learn. And that I know how it works, I can make any app that needs realtime information update, like I did with the next app - A Two-Player Tic-Tac-Toe game. I will try to add file sharing feature next.

Tic-Tac-Toe

This app also uses Websocket. The best thing about this app is that it does not need any database. It uses stores informations in an In-Memory storage which I created myself using mere HashMap. The game logic and user interaction part took the most brainpower to implement. But at the end I did it. Yay!!!
