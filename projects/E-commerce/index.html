E-commerce Model

This is the first project I started my Spring Boot journey with.

This is a model of an E-commerce website. I challenged myself to build a Spring Boot project without watching any tutorial. So I learned everything from Spring's official documentation. Let me show you what I have built.

It is a simple watch shop. I had created a similar project with Python's Flask earlier. I borrowed some ideas from there. I screapped this website with the help of Python's BeautifulSoup4 library to get Names and Images of a few watches. I saved those details in MySQL database using basic JDBC. When the "Add to cart" button is clicked the watch is added to the cart. The number of items can be increased and decreased and removed. These functionality are implemented by JavaScript's fetch api.

Once you are ready you can click the "Get Receipt" button. And the app will send you the bill as pdf. I used iTextPdf library for this purpose.

But wait. There is more.

There is also an admin panel where admin can add new items. This is not a real website. So I am not going into editing product part.
Also it generates sales report as excel file. I used Apache Poi to implement it.

And I added an About page for no reason at all.
